How it works

Add four lines to your agent. That's it.

your-agent.js 
const monitor = require('clawsight');
const agent = monitor({ server: 'https://app.clawsight.org' });

agent.init({ id: 'my-bot', name: 'Trade Bot' });
agent.log('Scanning markets...');  // shows up instantly
